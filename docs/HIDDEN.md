# HIDDEN.md

이 문서는 `tao logue`의 숨겨진 아이템 후보를 브레인스토밍하고 관리하기 위한 기획 문서다.

확정된 아이템 목록이 아니라 후보 풀로 사용한다. 충분히 많은 후보를 뽑은 뒤, 최종적으로 11개를 골라 각 월 또는 주요 진행 구간에 배치한다.

## 기본 원칙

- 숨겨진 아이템은 플레이어에게 존재를 명시하지 않는다.
- 캐릭터 대사, 배경 묘사, 어색한 연출, 반복 플레이 패턴으로만 암시한다.
- 일반 전투 보상보다 강하거나 고유해야 한다.
- 특정 빌드나 플레이 스타일을 지원해야 한다.
- 일부 아이템은 혼돈치를 증가시킬 수 있다.
- 일부 아이템은 강력한 대신 조건, 미니게임, 선택 리스크를 가진다.

## 역할 분류

숨겨진 아이템은 아래 역할 중 하나 이상을 가진다.

- 행동 횟수 증가
- 기본 강화 상태 시작
- 이동 보조
- 드로우 보조
- 혼돈 제어
- 엑스트라 덱 강화
- 족보 조건 보조
- 보스 정보 획득
- 특정 카드 타입 강화
- 리스크를 대가로 한 강한 성장

## 현재 아이디어

### 행동 횟수 관련

- 행동 횟수를 1 늘리는 아이템.
- 행동 횟수를 직접 늘리지는 않지만, 매 전투 첫 카드가 강화 상태로 시작하는 아이템.
- 강화 카드를 사용하지 않아도 첫 공격 또는 첫 방어가 강화되는 아이템.

### 간장과 막야

간장과 막야를 서로 배타적인 숨겨진 아이템으로 둘 수 있다.

- 간장: 공격적 선택지. 물리 공격, 반격, 큰 데미지 빌드와 연결.
- 막야: 안정적 선택지. 방어, 강화, 회복, 도술 유틸과 연결.

두 아이템은 하나만 선택하게 하거나, 서로 다른 월/조건에서 발견되게 할 수 있다. 둘 다 획득할 수 있게 할 경우 강한 족보나 특수 이벤트와 연결하는 방향도 검토한다.

```text
비고:
```

## 최종 선정 기준

최종 11개를 고를 때 아래 기준을 사용한다.

- 특정 타입이나 빌드에 과하게 몰리지 않는가?
- 행동 횟수 증가처럼 너무 강한 효과가 과도하게 많지 않은가?
- 혼돈치와 연결되는 아이템이 충분한가?
- 엑스트라 덱을 매력적으로 만드는 아이템이 있는가?
- 숨겨진 아이템답게 발견했을 때 기억에 남는가?
- 월별 테마와 연결할 수 있는가?

## 미정 사항

- 최종 아이템 수를 11개로 고정할지 여부
- 1월에도 숨겨진 아이템을 둘지 여부
- 간장/막야를 선택형 아이템으로 둘지 여부
- 행동 횟수 증가 아이템의 리스크
- 기본 강화 상태 시작 아이템의 구체 조건
